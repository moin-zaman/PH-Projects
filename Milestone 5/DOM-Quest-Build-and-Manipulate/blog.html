Important FAQ

What is the DOM in JavaScript?
The DOM (Document Object Model) is a structured representation of an HTML document that allows JavaScript to manipulate elements dynamically.
How do you select an element by its ID in the DOM?
Use document.getElementById("elementID") to select an element by its ID.
How can you change the text inside an HTML element using JavaScript?
Use element.innerText = "New Text"; to modify the text inside an element.
